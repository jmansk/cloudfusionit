🛠 Utility Classes

  • .text-center – Center text inside any container
  • .container – Adds horizontal margins and responsive width
  • .bg-light – Light gray background for visual grouping
  • .btn – Primary blue button used across site
  • .cta – Call-to-action styling for attention-grabbing links
  • .card:hover – Subtle movement on hover for visual feedback
